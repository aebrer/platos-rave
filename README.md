# Plato's Rave

An incremental idle game with clicking mechanics, inspired by the legendary 54-minute
uninterrupted bit from [MBMBaM Episode 674](https://maximumfun.org/episodes/my-brother-my-brother-and-me/mbmbam-674-platos-rave/).

Progress through 10 rooms of escalating absurdity hidden inside a Container Store.
Click to build Vibe. Unlock automation. Survive the traps. Eat the wings. Dethrone the
kandi king. Become a franchisee. Do it all again.

## Play

Coming soon on [itch.io](https://aebrer.itch.io).

## Development

Vanilla HTML/CSS/JS — no framework, no build step. Open `docs/index.html` in a browser.

For local development with a server:
```bash
cd docs && python3 -m http.server 8080
```
