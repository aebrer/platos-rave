# Plato's Rave — Game Design Document

## Overview

**Genre**: Incremental idle game with clicking mechanics
**Platform**: Web (mobile-first, desktop inherited)
**Theme**: 10-room rave hidden inside a Container Store, escalating absurdity
**Source**: MBMBaM Episode 674 (see `reference/episode-notes.md`)

---

## Core Loop

```
Click for Vibe → Unlock room generators → Automate Vibe income
→ Progress through rooms → Reach Room 10 → Prestige
→ Earn Kandi → Buy permanent upgrades → Start again stronger
```

### Primary Currency: Vibe
- Generated by clicking (base rate)
- Generated passively by room generators (automation)
- Spent to unlock rooms, buy upgrades, and power generators
- Resets on prestige

### Meta Currency: Kandi
- Earned by completing a full Room 10 cycle (prestige)
- Amount scales with total Vibe earned during the run
- Persists across prestiges
- Spent on permanent upgrades, cosmetics, and absurd unlocks

---

## Rooms

Each room is a "stage" the player progresses through. Unlocking a room costs Vibe and
adds a new generator to the player's income. Each room has:
- A Vibe threshold to unlock
- A unique generator with upgradable output
- Flavor text and visual theming from the episode
- Potential special mechanics

### Room 1 — The Container Store
*"Welcome to the Container Store. You don't know it yet, but you're already at the rave."*
- Entry point. Cheapest generator.
- Generator: Organize Bins (click containers into neat rows)
- Mechanic: Tutorial / introduction

### Room 2 — The Cloister of Confidence
*"You must be confident enough to risk trespassing charges to find the passage."*
- Generator: Sneak Past Employees (confidence builds Vibe)
- Mechanic: First meaningful automation unlock

### Room 3 — Dry Rub Wings
*"Buffalo Wild Wings, DoorDashed in. Dry rub only."*
- Generator: Eat Wings (dry rub fuels the vibe)
- Mechanic: First snack checkpoint. Travis's snack checkpoint system.

### Room 4 — The Escalation
*"The bass is getting louder. The lights are getting stranger."*
- Generator: Turn Up the Bass (rave intensity increases)
- Mechanic: Vibe multiplier mechanic introduction

### Room 5 — Barometric Pressure
*"Is it underground? Underwater? Purely psychological?"*
- Generator: Pressurize (withstand the crushing vibe)
- Mechanic: Some kind of pressure/risk mechanic — generators here
  produce more but with variance?

### Room 6 — Sauce on Six
*"Buffalo Wild Wings again, but now with SAUCE."*
- Generator: Sauce Wings (sauce amplifies everything)
- Mechanic: Second snack checkpoint. Sauce multiplies Room 3 output too.

### Room 7 — Traps (feat. Trapt)
*"Actual traps. Also the band Trapt plays here. Staind-brand earplugs available."*
- Generator: Dodge Traps (survive the gauntlet)
- Mechanic: Risk/reward — traps can temporarily reduce income but
  successful dodges give burst Vibe. Active clicking minigame?

### Room 8 — Second Cloister of Confidence
*"The brother of the employee from Room 2 confronts you."*
- Generator: Assert Dominance (you've been here before, but harder)
- Mechanic: Mirror of Room 2 but with escalated costs/rewards

### Room 9 — The Final Feast
*"Third snack checkpoint. You'll need the energy for what's next."*
- Generator: Power Up (final preparations)
- Mechanic: Buffs for the Room 10 encounter

### Room 10 — The Throne
*"A single person on a throne wearing a kandi crown. Only one at a time."*
- No generator — this is the prestige trigger
- The kandi king assesses your party level
- **"Not yet"** → Sent back to Room 1 (fail condition if Vibe too low?)
- **Dethrone** → Prestige. Earn Kandi. You become the king. A new rave begins.

## Prestige System

### The Cycle
Reaching Room 10 with enough accumulated Vibe triggers prestige:
1. All Vibe resets to 0
2. All room generators reset
3. Player earns Kandi based on total Vibe earned during the run
4. Player restarts at Room 1

### Kandi Upgrades (Permanent)
Persist across prestiges. Examples:
- **Kandi Bracelet**: +X% base click value
- **Glow Sticks**: +X% all generator output
- **VIP Wristband**: Start with Room N already unlocked
- **Better DoorDash**: Wings rooms produce X% more
- **Staind Earplugs**: Room 7 traps deal less damage
- **Extra Keys**: 300-key system activates faster (global multiplier event)
- **Franchise License**: +X% Kandi earned per prestige

### Cosmetics / Absurd Unlocks
- Hats for the kandi king (visible on prestige screen)
- Room reskins (different Container Store locations)
- Alternative wing flavors
- Different bands in Room 7 (but Trapt is always there too)

## Offline Progression

### Catch-Up Mechanic
When the page loads:
1. Read last save timestamp from LocalStorage
2. Calculate elapsed seconds since last save
3. Simulate production: `elapsed_time * total_passive_income`
4. Apply cap (e.g., max 24 hours of offline earnings, configurable)
5. Show "Welcome back!" summary of what was earned offline

### Why This Matters
- Cookie Clicker requires the tab to stay open — we don't
- Respects the player's time and attention
- Encourages short check-in sessions on mobile

---

## 300-Key Drop Event

Periodic event inspired by the 300-key simultaneous activation system:
- A progress bar fills over time (e.g., every 5 minutes)
- When full, the "THX buildup" begins
- Player spam-clicks during the buildup window
- On completion: massive temporary Vibe multiplier
- Upgradeable: more keys = faster fill, bigger multiplier, longer duration

This gives the clicker side real engagement beyond buying upgrades.

## Save System

- **Storage**: `localStorage.setItem('platos-rave-save', JSON.stringify(state))`
- **Auto-save**: Every 30 seconds + on page unload (`beforeunload`)
- **Schema version**: Integer field for future migrations
- **Manual export/import**: JSON string copy/paste (future feature)

### Save State Structure (v1)
```json
{
  "version": 1,
  "lastSaved": 1710000000000,
  "vibe": 0,
  "totalVibeEarned": 0,
  "kandi": 0,
  "prestigeCount": 0,
  "rooms": {
    "1": { "unlocked": true, "level": 1 },
    "2": { "unlocked": false, "level": 0 }
  },
  "upgrades": {},
  "stats": {
    "totalClicks": 0,
    "totalVibeAllTime": 0,
    "fastestPrestige": null
  }
}
```

## V1 Proof of Concept Scope

The first playable build should have:
- [x] Basic HTML/CSS/JS structure (mobile-first)
- [ ] Click-to-earn Vibe mechanic
- [ ] Room 1 and Room 2 unlockable with generators
- [ ] Passive income from generators
- [ ] LocalStorage save/load with offline catch-up
- [ ] Auto-save on interval + page unload
- [ ] Minimal but functional UI showing Vibe count, generators, room progress
- [ ] Room unlock animations/transitions

**Not in v1:** Prestige, Kandi, Rooms 3-10, 300-key event, cosmetics, asset generation.

15. Room flavor text and basic theming
16. "Welcome back" offline earnings popup
17. Responsive layout testing on mobile
18. QA checklist pass
